# Netzwerk-Einstellungen

Wenn Impressum Plus netzwerkweit in einer Multisite aktiv ist (was empfohlen wird), kannst du Standardwerte unter **Netzwerk > Einstellungen > Impressum** setzen. Diese Werte werden dann verwendet, wenn keine anderen  – spezifischeren – Daten auf der einzelnen Website gesetzt wurden. Auf diesem Weg kannst du automatisch identische Daten innerhalb deines Netzwerks vorgeben, wenn gewünscht.

Außerdem solltest du, wenn netzwerkweit aktiv, deine API und Lizenzdaten hier verwalten.
