# Feldabhängigkeiten <Badge type="tip" text="Nur in Form Block Pro" />

Für jedes Formularelement kannst du festlegen, dass es von anderen Feldern abhängig ist. Auf diese Weise wird es ausgeblendet, solange nicht alle definierten Bedingungen erfüllt sind. Du kannst beispielsweise das Feld „Telefonnummer“ ausblenden, solange niemand das Feld „Rückruf anfordern“ angekreuzt hat, da die Telefonnummer nur dann wichtig ist, wenn der Benutzer einen Rückruf wünscht.

Wenn ein Formularelement von mindestens einem anderen Feld abhängig ist, wird es im Editor mit einem roten gepunkteten Rahmen versehen.
